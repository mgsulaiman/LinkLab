## What is LinkLab?

LinkLab is a lightweight macOS menu bar app that lets you manage, test, and automate deeplinks on the iOS Simulator. No more copy-pasting URLs into Terminal. Add your deeplinks once, organize them by app, and test with a single click.

## Features

### Deeplink Management
- Add, edit, delete, and reorder deeplinks with drag-and-drop
- Search and filter across all your deeplinks
- Smart URL autocomplete from existing entries
- Copy any deeplink URL to clipboard instantly
- Generate QR codes for any deeplink on-demand

### Three Test Modes
- **Normal** — Opens the app (if needed) and triggers the deeplink
- **Cold Start** — Force-kills the app, then triggers the deeplink to test fresh launch behavior
- **Warm Start** — Backgrounds the app, then triggers the deeplink to test resume behavior

### Smoke Testing
- Run **all** deeplinks automatically through your selected test modes
- Individual deeplink testing with status indicators
- Configurable time delays between tests

### Automatic Recording & Screenshots
- Capture MP4 video recordings of every deeplink test
- Automatic screenshots at configurable delays
- Files organized by app name, date, and test mode
- Customizable output folder

### Multi-App & Environment Support
- Manage deeplinks for **multiple apps** in one place
- Separate **production** and **development** bundle identifiers per app
- One-click environment switching
- Optional app termination after each test

### Simulator Management
- View all simulators with boot/shutdown status
- Boot, shutdown, erase, clone, rename, and delete simulators
- Create new simulators with custom device types and runtimes
- Search and filter by name or iOS version

### Import & Export
- Export deeplinks as JSON (includes app bundle metadata)
- Import deeplinks with smart conflict handling (skip or overwrite duplicates)
- Auto-creates app bundles from imported JSON metadata
- Backwards-compatible with plain deeplink arrays

### Privacy & Security
- **100% local** — all data stays on your Mac
- No analytics, no tracking, no telemetry
- No third-party SDKs
- Works fully offline
- Enterprise-safe: proprietary URLs never leave your machine

## Requirements

- macOS 13.0 or later
- Xcode with iOS Simulator installed
